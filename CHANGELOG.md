# Changelog

## 0.1.3 - 2026-05-05

- D88生成時の既定動作を「新規作成」に変更
- `--d88-append-if-exists` を追加し、既存D88へ追加する挙動を明示指定できるように
- GUI の `既存D88ファイルがある場合はファイルを追加` チェックに対応

## 0.1.2 - 2026-05-05

- `mz2500-tools` の `mzd88.rb` を取り込み、D88ディスクイメージ生成を統合
- `--d88-sidecar keep|delete` を追加し、D88追加後の `BRD` / `BSD` 保持を選択可能に
- D88追加順を `BRD` → `BSD` に修正
- 512色モードのファイル命名を `_FR` / `_FG` / `_FB` に短縮
- `split320x200` の結合BSD名を `_c.bas.bsd` に変更
- D88内部ファイル名の安全側制限を 16 bytes として扱うよう調整
- 4096色 `.palette` にタイトル・列見出し・使用数ヒストグラムを追加

## 0.1.1 - 2026-04-24

- 512色モードの固定チャンネル解釈を実機確認に合わせて修正
- 固定チャンネルの段を `1/3/5/7` から `0/2/4/6` に変更
- 512色モードのPNG保存時に不要な追加シミュレーションを行わないよう修正
- README を整理し、導入手順と説明を改善

## 0.1.0-alpha - 2026-04-23

- 初回公開版
- GUI連携向けの `--info --json` とバージョン出力を追加
- JPEG入力対応
- リサイズモード `fit` / `keep` / `cut` を追加
- `--png-only` を追加
- `.palette` 出力と README の公開向け整備を実施
