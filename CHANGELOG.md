# Changelog

このファイルでは、`pngconvMZ` の主な変更履歴を記録します。

## 0.1.1 - 2026-04-24

- 512色モードの固定チャンネル解釈を実機確認に合わせて修正
- 固定チャンネルの量子化段を `1/3/5/7` から `0/2/4/6` に変更
- 512色モードのPNG保存時に不要な追加シミュレーションを行わないよう修正
- README を整理し、導入手順と依存関係の説明を改善

## 0.1.0-alpha - 2026-04-23

- 初回公開版
- GUI連携向けの `--info --json` とバージョン出力を追加
- JPEG入力に対応
- リサイズモード `fit` / `keep` / `cut` を追加
- `--png-only` を追加
- `.palette` 出力やREADMEを含む公開向けの整理を実施
